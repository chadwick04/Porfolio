User Interface Design

Features of the Project

  • Login – where you input your username and password.
  • Main – where you can see three buttons and pick which one you open first.
  • Guess Number – where you input a number between 1-5 only and try to guess the random number the computer give.
  • Math Question – where you are given to answer a four simple math problem.
  • Converter – where you input an hour to convert into minutes and seconds.
